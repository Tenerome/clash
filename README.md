## 订阅失败
主要是因为raw.githubusercontent.com域名解析失败,可以使用域名查询网站查询实时有效的DNS:[ip38查询](https://site.ip138.com/)  
然后修改hosts文件，windows下在`C:\Windows\System32\drivers\etc\hosts`,linux在`/etc/hosts`

##  直接使用原文档导入
实在订阅不了，就直接在`/res/`目录下，找最新的日期源文档，直接复制导入，其中的`*.out`文件是base64编码后的
